# 🛍️ Product & Collection UI – Internship Assignment

This project is built as part of a frontend internship assignment. It showcases a responsive product listing, filtering, sorting logic, and individual product detail view using **React**, **Vite**, and **Tailwind CSS**.

---

## 🚀 Tech Stack

- ⚛️ React (with Vite)
- 🎨 Tailwind CSS
- 🔁 React Router DOM (for routing)
- 📦 Mock data (no backend or DB)
- ♻️ Modular & reusable components

---

## 🧱 Features

✅ Product Grid/List View  
✅ Filter Sidebar (Responsive, Mobile-Friendly)  
✅ Product Detail Page with Add to Cart UI  
✅ Reusable Components:  
&nbsp;&nbsp;&nbsp;&nbsp;• `ProductCard`  
&nbsp;&nbsp;&nbsp;&nbsp;• `PriceTag`  
&nbsp;&nbsp;&nbsp;&nbsp;• `RatingStars`  
&nbsp;&nbsp;&nbsp;&nbsp;• `FilterSidebar`

---

## 📱 Responsive Design

- Works seamlessly across all screen sizes
- Mobile-first layout with adaptive sidebar and grid

---

## 🔨 How to Run Locally

```bash
# Clone the repo
git clone https://github.com/your-username/product-ui.git

# Navigate into the folder
cd product-ui

# Install dependencies
npm install

# Run the dev server
npm run dev
```
